# ADU Reports

## Installation

-   Clone this repository

-   Run `composer install`

-   Run `npm install`

-   Run `php artisan serve`

-   Run `npm run dev`

-   Setup your local database and add credentials to .env file.

-   Run `php artisan migrate`
